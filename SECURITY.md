# Security Policy

## Supported Versions

This is a static, client-side widget with no runtime dependencies and no
backend of its own. Only the latest `main` branch is supported; there are no
maintained older versions.

## Reporting a Vulnerability

If you find a security issue (for example an XSS vector, a build script
issue, or a dependency vulnerability), please report it privately by emailing
security@peptidesdirect.io (or support@peptidesdirect.io if you do not get a
response). Please do not open a public issue for security reports.

Include as much detail as you can: affected file(s), reproduction steps, and
the impact you expect. We will acknowledge your report and follow up as we
investigate.

There is no bug bounty program for this repository.
